Para dejar esta línea divisora hay que usar "hr", ahora tengo que hacer el punto 2, que es hacer una lista anidada de enlaces, al hacer click en cada enlace, debe llevarme a la misma página a la cual está dirigida.

Para hacer una lista de objetos, se puede utilizar "ol" para hacer una lista enumerada, y "ul" para hacer listas con puntitos, para hacer items de listas hay que utilizar "li" para cada item.

Por ejemplo:
    Temones
    • Opening de Evangelion
    • Bad Apple!
    • End Of The World Mad AMV
    Temones pero normales
    • Falling Through The Sky
    • Dissolution
    • Violent Purple

Todo esto se hace mediante el uso normal de las listas, para colocar los links se coloca la letra "a", en la sección de "href" hay que colocar el link al cual queremos dirigir, y dentro de los mismos corchetes del link se coloca "target=_blank" para que el link se abra en una pestaña nueva.
